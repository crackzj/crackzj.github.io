## 整数

- `python`可以处理“任意”大小的整数，包括负数

- 支持二进制、八进制、十进制、十六进制

```python
a = 0b1010
b = 0o12
c = 10
d = 0xa
```

| 前缀  | 例子     | 进制 |
| ----- | -------- | ---- |
| 0b/0B | a=0b1010 | 2    |
| 0o/0O | b=0o12   | 8    |
| 无    | c=10     | 10   |
| 0x/0X | d=0xa    | 16   |
